Lab-06-A

Start

Objectives

  • Explore the BlueJ diary-prototype project as a prelude to unit testing.
    • Project comprises three classes:
      • Week
      • Day
      • Appointment
  • Explain and demonstrate regression testing.
  • Provide an introduction to unit testing explaining roles of
    • Positive testing
    • Negative testing

Lab-06-B

Start

Objectives

  • Implement some simple unit tests on the diary-prototype project.

    • Manual tests by creating individual Appointment and Day objects and invoking methods on these objects.
    • Semi-automated tests by writing a DiaryTester class containing various test methods.
    • Introduce test automation showing how batches of unit tests can be run automatically.

  • Use BlueJ Debugger to locate any errors discovered during testing.

  • Demonstrate automated checking of test results.

  • Use the BlueJ feature to record and replay a suite of tests.

Description of a Java Interface

  • A Java type comprising group of public method signatures
  • May also include constant declarations that are static public by default
    • Methods have no implementation in interface definition
    • Implementation of interface methods takes place in classes that implement interfaces
    • Interface only specifies behaviour
      • Cannot create objects of an interface
      • Can create objects of classes that implement interfaces
    • All interface methods are public
    • All constant declarations are public static

Purpose of interfaces

  • To unify common behaviour

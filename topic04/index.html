Lab-04-A

Start

Objectives

  • Use BlueJ Notebook class to gain basic experience with the ArrayList collection class.
  • Explore some methods of ArrayList such as add, size, get, remove, clear.
  • Write a method to generate an ArrayList of test data to avoid keying in such data repeatedly.
  • Write and test methods to process the ArrayList using
    • for-each
    • while
    • iterator
  • Write and test a method to search for a given string as a subset of any of the elements in the Notebook collection, notes.
  • Write methods to perform the following operations on the Notebook collection:
    • remove specific individual element.
    • remove all elements.
